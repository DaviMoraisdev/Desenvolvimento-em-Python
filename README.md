# Fundamentos de Python

Repositório criado para registrar os exercícios e projetos desenvolvidos durante meus estudos de fundamentos da linguagem Python.

## Conteúdos estudados

Os exemplos deste repositório abordam:

- Variáveis e tipos de dados
- Entrada e saída de dados
- Operadores aritméticos e lógicos
- Strings e fatiamento
- Condicionais com `if`, `elif` e `else`
- Estruturas de repetição com `for` e `while`
- Listas, tuplas, conjuntos e dicionários
- List comprehension
- Funções e parâmetros
- `*args` e `**kwargs`
- Funções lambda
- Funções recursivas

## Projeto final do módulo

O arquivo `PythonFundamentos/Exercício_FinalMódulo1.py` contém um sistema executado pelo terminal para gerenciamento de times e jogadores.

O programa permite:

- Adicionar e remover times
- Listar os times cadastrados
- Adicionar jogadores a um time
- Remover jogadores de um time
- Listar os jogadores de cada time
- Exibir a quantidade de jogadores por time

## Como executar

É necessário ter o Python 3 instalado.

Clone o repositório:

```bash
git clone https://github.com/DaviMoraisdev/python-fundamentos-obc.git
cd python-fundamentos-obc
```

Execute o projeto final:

```bash
python "PythonFundamentos/Exercício_FinalMódulo1.py"
```

Em alguns sistemas, o comando pode ser:

```bash
python3 "PythonFundamentos/Exercício_FinalMódulo1.py"
```

Os demais exemplos podem ser executados da mesma maneira, informando o caminho do arquivo desejado.

> O arquivo `Exercicios.py` utiliza o módulo `winsound`, disponível somente no Windows.
